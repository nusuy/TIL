# 시각화

## 데이터 형식

- 정형 데이터 (데이터베이스, 스프레드시트, CSV)
- 반정형 데이터 (XML, HTML)
- 비정형 데이터 (이미지, 동영상, 음성, 텍스트)

> 4가지 데이터 형태 기억
>
> - 수치 데이터 (양적 데이터)
>   - 연속형 데이터 Continuous (키, 몸무게, 시간, 매출액) / 이산형 데이터 Discrete (수량)
> - 범주형 데이터 (질적 데이터) Categorical Data
>   - 순위형 데이터 Ordinal / 명목형 데이터 Nominal (성별, 지역)

> 레퍼런스 참고 시 버전 주의
